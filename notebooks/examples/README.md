# Example Notebooks

This directory contains worked example notebooks for each course topic.

These notebooks are student-facing and are indexed by the AI tutor.

## Naming convention

`NN-topic-name.ipynb` — where `NN` matches the chapter number.

Example: `01-introduction.ipynb`, `04-dataframes.ipynb`
